== What Is This?

Ephapaxiser enforces **linear type semantics** on resources in existing code —
file handles, network connections, crypto keys, database transactions — ensuring
each is used **exactly once** and properly consumed.

Ephapax (_ἐφάπαξ — once for all_) is a linear type system targeting WASM.
Ephapaxiser brings its guarantees to existing Rust, C, and Zig codebases.

== How It Works

Annotate resources in `ephapaxiser.toml`. Ephapaxiser:

1. Identifies resource acquisition and release patterns in your code
2. Generates Ephapax linear type wrappers that enforce single-use semantics
3. **Compile-time errors** if a resource is used after consumption, leaked, or double-freed
4. Zero runtime overhead — proofs erased at compile time, targets WASM

== Key Value

* **Resource leaks become compile errors** — files always closed, keys always zeroised
* **Use-after-free impossible** — linear types structurally prevent it
* **Crypto key safety** — keys used exactly once then destroyed (perfect forward secrecy)
* **Existing code wraps safely** — gradual adoption, one resource type at a time

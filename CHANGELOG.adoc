== [0.1.0] - 2026-03-21

=== Phase 1 Complete

- Resource handle detection in source files
- Ephapax wrapper generation with linear type enforcement
- Linearity analysis detecting leak, double-free, and use-after-free
- Multi-language codegen stubs (Rust Drop, C cleanup attrs, Zig defer)
- Bespoke manifest parsing for resource specification
- Integration tests for detection, wrapping, and linearity checks

=== Added (scaffold — 2026-03-20)

- Initial project scaffold from rsr-template-repo
- CLI with subcommands (init, validate, generate, build, run, info)
- Manifest parser (`ephapaxiser.toml`)
- Codegen engine stubs
- ABI module (Idris2 proof type definitions)
- Library API for programmatic use
- Full RSR template (17 CI workflows, governance docs, bot directives)
- README.adoc with architecture overview
